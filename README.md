# Library Management DevOps Project

## Technologies Used

- Flask
- MySQL
- Bootstrap
- Docker
- Docker Compose
- Jenkins
- GitHub

## Features

### Admin
- Admin Login
- Add/Delete Books
- Add/Delete Students
- Allocate Books
- Return Books
- Dashboard Analytics

### Library Functions
- Availability Tracking
- Automatic Quantity Update
- Borrow Management
- Student Management

## Run Project

docker-compose up --build

## Open

http://localhost:5000
